VScode和WSL搭建C开发环境

  1. 首先安装wsl，微软商店就有。

  2. 安装VScode。

  3. 换源

    sudo vim /etc/apt/source.list

  4. 更新ubuntu软件，sudo apt-get update

  5. 安装gcc，gdb等工具

  6. 在VScode中安装wsl扩展插件。

  7. 在VScode中安装C/C++插件。

  8. 安装code runner插件。

  9. 要将VScode的目录bin添加到环境变量

  10. code runner插件中有run in terminal

  11. 设置文件执行命令，文件生成目录，在code runner插件的executor map中设置。
